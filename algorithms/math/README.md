# Sieve of Eratosthenes
Sieve of Eratosthenes is an algorithm for finding all the prime numbers in a segment [1:n]

## Algorithm Analysis
```
N is the upper bound of the range

Time Complexity:    O(N * loglogN)
Space Complexity:   O(N)
```
